# Extensions Creator

This extension assists in creating new Gemini CLI extensions.

## Usage

Run the following command to start the scaffolding wizard:

```bash
/new-extension
```

This will activate the `extension-expert` skill and guide you through:
1.  Naming and describing your extension.
2.  Setting up the directory structure.
3.  Creating the `gemini-extension.json` manifest.
4.  Adding tools, skills, or commands.
