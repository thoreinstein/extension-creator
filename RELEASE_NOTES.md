# v1.0.0 - Initial Release

## Summary
Initial release of the `extensions-creator` extension. This tool streamlines the process of building new Gemini CLI extensions by providing an expert scaffolding wizard and a comprehensive reference library.

## New Features
- **Extension Wizard**: New `/new-extension` command to scaffold extension structure.
- **Extension Expert Skill**: A specialized agent skill (`extension-expert`) equipped with official documentation on writing extensions, manifests, skills, hooks, and MCP servers.
- **Reference Library**: Bundled local documentation for offline reference.
